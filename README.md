# LamportClocks

> Authors: Gabriel Castro, Gustavo Possebon e Henrique Kops

_Java implementation over Lamport's clocks._

### How to run

At project's root:

1. Package code into jar with:

    ```
    $ mvn clean package
    ```
1. Execute with java:

    ```
    $ ./run.sh <confFilePath: str> <currentNodeId: int>
    ```
